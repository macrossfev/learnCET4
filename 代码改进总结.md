
### 2. 常量配置 `miniprogram/utils/constants.js`
```javascript
module.exports = {
  TOTAL_WORDS: 1000,
```

### 3. 通用工具 `miniprogram/utils/common.js`
提供以下函数：
- `shuffle(arr)` - Fisher-Yates 洗牌算法
- `getRandomItems(arr, count)` - 随机获取元素
- `getToday()` - 获取今日日期
- `calcReviewDates(fromDate, intervals)` - 计算复习日期
- `safePercent(part, total, decimals)` - 安全百分比计算
- `log(level, message, data)` - 统一日志输出
- `debounce(fn, delay)` - 防抖函数
- `throttle(fn, interval)` - 节流函数

### 4. 错误处理 `miniprogram/utils/error.js`
提供以下函数：
- `handleError(err, defaultMessage, options)` - 统一错误处理
- `safeAsync(fn, errorMessage)` - 安全执行异步函数
- `showLoading(title)` - 显示加载提示
- `hideLoading()` - 隐藏加载提示
