## 第一阶段：数据准备（已完成）

### 完成日期：2026-03-06

### 已完成工作

#### 1. 词频数据采集
- 数据来源：GitHub PigWord/HashTable_dictionary/CET4.csv（含CET-4真题词频、音标、释义）
- 原始数据：6701条记录
- 处理流程：过滤停用词 → 去除试卷指令词（section/passage/blank等） → 与CET-4大纲词表取交集 → 按真题频次降序排列
- 最终产出：3393条有效词条

#### 2. 词条数据生成
- 音标（美式）：3383/3393 已填充（99.7%）
- 中文释义：3388/3393 已填充（99.9%）
- 英文词组：3393/3393 已填充（100%）—— 通过并行Agent批量生成
- 英文例句：3393/3393 已填充（100%）—— 通过并行Agent批量生成
- 词组/例句中文翻译：全部已填充

#### 3. 工具脚本

| 脚本 | 路径 | 功能 |
|------|------|------|
| freq_analysis.py | data/scripts/ | spaCy词频分析（如有新真题文本可重新分析） |
| generate_words.py | data/scripts/ | 词条模板生成（--mode template）/ 合并校验（--mode merge） |
| generate_audio.py | data/scripts/ | Edge TTS批量生成音频，支持断点续跑、并发控制 |
| fill_word_data.py | data/scripts/ | LLM API批量填充phrase/example（备用，本次已直接完成） |

### 关键技术决策
- 前端：微信小程序原生开发（JS基础版模板）
- 后端：微信云开发（MongoDB + 云函数 + 云存储）
- 音频：Edge TTS批量预生成，美音JennyNeural，中文XiaoxiaoNeural
- 发音：美音
- 词汇范围：CET-4高频前1000词
- 每日默认20词，可调
- 复习间隔：1/3/7/14/30天，不主动提醒，进入时显示
- 播放顺序：英文单词 → 中文释义 → 英文词组 → 词组中文 → 英文例句 → 例句中文
- 详细设计见 开发文档.md
